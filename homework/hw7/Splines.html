Curve-Editor (Bezier)

Add point: click mouse in an empty space.

Delete latest added point: Press "X" key

Delete selected point: Press "D" key while the cursor is pointing to the point needed to be deleted.

Move a point: press the mouse while the cursor is pointing to the point and simply move the cursor.

Auxiliary Line:

Input Coordinates

Example: {"x":-1,"y":0,"z":0}, {"x":0,"y":1,"z":0}, {"x":1,"y":0,"z":0}
